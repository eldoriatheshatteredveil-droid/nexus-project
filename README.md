# NEXUS: The Digital Frontier

Welcome to NEXUS, a futuristic cyberpunk indie game storefront built with **Vite, React 19, and TypeScript**. This project aims to provide a free, open alternative to platforms like itch.io for indie game developers and players.

> **Note:** This project was built with the assistance of AI games.

## Features

- **Dynamic Game Listings**: Browse through a wide variety of indie games.
- **Game Details**: View detailed information about each game, including descriptions and play options.
- **User Authentication**: Secure sign-in and sign-up functionality.
- **Upload Protocol**: Users can upload their own games (browser-based or downloadable).
- **Custom Aesthetic**: A unique dark mode design with glassmorphism effects and smooth scrolling.

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/eldoriatheshatteredveil-droid/nexus-project.git
   cd nexus-project
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## Deployment

To build the project for production, run:
```bash
npm run build
```
Then, you can deploy the `dist` directory to your preferred hosting service (e.g., Netlify).
